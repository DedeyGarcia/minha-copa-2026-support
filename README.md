# Minha Copa 2026 - Suporte

Bem-vindo à página de suporte do **Minha Copa 2026**. 
Este aplicativo foi desenvolvido para ajudar colecionadores a gerenciarem suas figurinhas do álbum físico de forma simples e rápida.

## 🛠 Como podemos ajudar?
Se você encontrou algum bug, tem dúvidas sobre o scanner de figurinhas ou sugestões de melhorias, utilize um dos canais abaixo:

* **E-mail:** andreylucas.garcia@gmail.com
* **GitHub Issues:** Você pode abrir uma "Issue" diretamente neste repositório relatando o problema.

## 🔒 Privacidade e Câmera
O app solicita acesso à câmera exclusivamente para a funcionalidade de escaneamento e identificação automática das figurinhas. 
* As imagens processadas **não são enviadas para servidores externos**.
* O processamento é feito localmente no seu dispositivo para garantir sua privacidade.
